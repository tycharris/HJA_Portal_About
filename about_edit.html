Project Summary

This project seeks to develop a web map-based data catalog for the H.J. Andrews Experimental Forest. The initial goal is to provide a central platform for researchers and the broader public to visualize geospatial data within a 3D digital landscape. The portal was developed using Cesium, an open-source JavaScript library for 3D maps, in conjunction with TerriaJS, a library for building geospatial data catalogs, and GeoServer, a Java-based software server that allows users to view and edit geospatial data. The datasets currently available are for demonstration purposes; this is a pilot project to be built upon in the future.

Background

The H.J. Andrews Experimental Forest is an NSF-funded Long-Term Ecological Research Site situated in the Western Cascades of Oregon. Since its establishment in 1948, the Andrews research program has generated a diverse and extensive catalog of geospatial datasets. Until now, there has been no central, web map-based repository for these datasets.


Future Directions

Considering the Andrewsâ€™ large and active research community, and the extensive catalog of data available, we see an opportunity to provide a web map service that promotes collaboration and improves understanding of the forest ecosystem. We plan to build on this proof of concept by adding all of the geospatial datasets available through the H.J. Andrews website to our catalog. Other near-term goals include visualization of time-series data products derived from a spectral trajectory-based change detection algorithm known as LandTrendr (developed by Dr. Robert Kennedy), high resolution data derived from LiDAR and UAV's, and real-time data from wireless sensors. Our primary long-term goal is to develop interactive, analytical widgets for on-the-fly summarization and charting of datasets.
